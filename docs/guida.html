📊 Smart Node Monitor

FastAPI Supabase Realtime IoT Simulation  — Guida completa al progetto

🎯 Il Progetto

Smart Node Monitor è un sistema full-stack per il monitoraggio remoto di sensori industriali in tempo reale. Simula uno scenario reale: un sensore di temperatura installato su un macchinario (es. frigorifero industriale per medicinali) che invia dati ogni 5 secondi a un backend cloud, il quale li salva su database e li espone tramite una dashboard web interattiva.

L'operatore può monitorare la temperatura da qualsiasi dispositivo, ricevere alert visivi quando supera soglie critiche e controllare attuatori remoti (LED, buzzer) direttamente dalla dashboard.

Il progetto dimostra competenze end-to-end: dalla simulazione hardware al deploy cloud, passando per API REST, database real-time e UI moderna.

🚀 Come Avviare

Aprire 3 terminali PowerShell separati ed eseguire in ordine:

1

Backend — FastAPI (porta 8000)

$env:PYTHONPATH = "C:\Users\aeria\Sviluppo\smart-node-monitor"
cd C:\Users\aeria\Sviluppo\smart-node-monitor\backend
..\venv\Scripts\uvicorn.exe app.main:app --reload --port 8000

Attendi: Application startup complete.

2

Frontend — Dashboard (porta 3000)

cd C:\Users\aeria\Sviluppo\smart-node-monitor\frontend
python -m http.server 3000

Poi apri nel browser: http://localhost:3000

3

Simulatore IoT — invia dati ogni 5 secondi

cd C:\Users\aeria\Sviluppo\smart-node-monitor
.\venv\Scripts\python.exe simulator\device_simulator.py

Vedrai: ✓ Dati inviati: 28.3°C ogni 5 secondi
