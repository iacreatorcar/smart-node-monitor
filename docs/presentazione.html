Overview

Cosa fa il progetto

Smart Node Monitor simula un nodo IoT che raccoglie dati ambientali (temperatura e umidità), li trasmette a un backend cloud e li visualizza in tempo reale su una dashboard web. L'operatore può controllare remotamente attuatori fisici — LED e buzzer — direttamente dal browser.

Il progetto dimostra un'architettura full-stack completa: dal dispositivo simulato (Python) fino all'interfaccia utente, passando per API REST, database cloud e WebSocket realtime.

📡

Acquisizione Dati Real-time

Simulatore Python invia dati ogni 5 secondi. Dashboard aggiorna automaticamente via Supabase Realtime (WebSocket).

🌡️

Monitoraggio Temperatura

Soglie visive: verde normale, arancio sopra 25°C, rosso critico sopra 30°C con alert banner.

🎛️

Controllo Attuatori

Toggle LED e Buzzer dal browser. Stato persistito su DB, sincronizzato su tutti i client aperti.

📺

Monitor TV Fullscreen

Vista ottimizzata per display esterni: temperatura gigante, grafico, stato attuatori. Aggiornamento istantaneo.

📈

Storico Grafico

Grafico Chart.js con ultime 50 letture. Colore dinamico in base alla temperatura massima rilevata.

☁️

Deploy Cloud

Backend su Render.com, frontend statico deployabile su Netlify. Database su Supabase (PostgreSQL).

Ispirazione e contesto

Su cosa si basa

Il progetto nasce dalla convergenza di due mondi in rapida crescita: l'Internet of Things industriale e le moderne architetture cloud-native. È stato costruito come progetto dimostrativo per un colloquio, con l'obiettivo di mostrare competenze full-stack concrete e applicabili in contesti reali.

Pattern di riferimento: il sistema segue l'architettura tipica dei sistemi SCADA (Supervisory Control and Data Acquisition) in versione moderna e cloud-based. Invece di hardware costoso e reti proprietarie, usa API REST standard e WebSocket su infrastruttura serverless.

Casi reali analoghi: sistemi di monitoraggio ambientale in data center, controllo temperatura in celle frigorifere industriali, supervisione remota di impianti fotovoltaici, monitoring di server farm.

Applicazioni

Casi d'uso reali

  • Monitoraggio temperatura in ambienti critici (server room, celle frigo)
  • Dashboard operativa per impianti industriali leggeri
  • Sistema di allerta precoce per variazioni ambientali
  • Controllo remoto attuatori in automazione domestica
  • Visualizzazione dati IoT su display pubblici o sala controllo
  • Prototipo per integrazione con hardware reale (ESP32, Raspberry Pi)
